# 터미널용 학식 메뉴 조회 봇

이 폴더는 터미널 환경에서 명령어 하나로 쉽게 학식 메뉴를 조회할 수 있도록 만들어진 프로그램입니다.

## ⚙️ 설치 및 세팅 방법

**1. conda 가상환경 구성**
터미널(Anaconda Prompt 또는 cmd)을 열고 현재 폴더(`terminal`)로 이동한 뒤, 아래 명령어를 실행하여 필요한 패키지가 포함된 가상환경을 생성합니다.

```cmd
conda env create -f environment.yml
```

**2. 환경변수 등록 (선택사항, 전역에서 'menu' 명령어를 쓰기 위함)**
cmd에서 현재 폴더가 아닌 어디서든 `menu`만 입력해서 학식 정보를 보려면, 지금 위치한 `terminal` 폴더의 경로를 윈도우 **시스템 환경 변수(PATH)** 에 추가해주시면 됩니다.

**[환경 변수 등록 상세 가이드]**
1. 윈도우 검색창(시작 버튼 옆)에 **"환경 변수 편집"** 또는 **"시스템 환경 변수 편집"**을 검색하여 실행합니다.
2. 시스템 속성 창이 열리면 우측 하단의 **[환경 변수(N)...]** 버튼을 클릭합니다.
3. 위쪽의 '**사용자에 대한 사용자 변수**' 또는 아래쪽의 '**시스템 변수**' 목록에서 **`Path`** 변수를 찾아 선택하고 **[편집(E)...]** 버튼을 누릅니다.
4. 우측의 **[새로 만들기(N)]** 버튼을 클릭합니다.
5. 현재 `terminal` 폴더의 전체 경로를 복사하여 붙여넣고 엔터를 누릅니다.
6. 열려있는 모든 창에서 **[확인]** 버튼을 눌러 설정을 저장하고 닫습니다.
7. 진행 중이던 터미널 창을 닫고, **새로운 cmd 창을 열어서** `menu`를 입력해 정상 실행되는지 확인합니다.

---

## 🚀 실행 방법

### 방법 1 (해당 폴더에서 실행)
터미널을 열고 본 스크립트가 있는 `terminal` 폴더로 이동한 뒤, 아래 명령어를 입력합니다.

```cmd
menu
```
그러면 자동으로 `menu` 가상환경이 활성화되고 학식 정보를 스크래핑해 출력합니다.

### 방법 2 (환경변수 PATH 등록 후 실행)
PATH에 이 폴더가 등록되었다면 윈도우의 어느 폴더에서나 단순히 아래와 같이 입력하면 됩니다.

```cmd
menu
```
